# Roadmap

Milestones describe capability slices, not release dates.

- **v0.1 Foundation:** repository governance, transactional installer core,
  lifecycle commands, baseline session, and CI.
- **v0.2 Hyprland:** monitor discovery UI, per-device input profiles, GPU-tested
  matrices, and configuration migration framework.
- **v0.3 Waybar:** responsive multi-monitor menu bar and accessible widgets.
- **v0.4 Dock:** dedicated native dock with autohide, magnification, running
  state, recent applications, and persistent drag reorder.
- **v0.5 Spotlight:** provider-based apps, files, calculator, clipboard, and unit
  conversion with privacy controls.
- **v0.6 Notification Center:** calendar, media, history, and coherent Control
  Center panels.
- **v0.7 Installer:** guided hardware choices, dry-run plan, mirrors, network
  recovery, and automated VM integration tests.
- **v1.0 Stable:** documented compatibility contract, accessibility audit,
  signed releases, upgrade migrations, and hardware validation.
