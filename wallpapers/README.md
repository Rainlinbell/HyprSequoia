# Wallpapers

Future wallpaper packs will be stored here with source and license metadata.
The default original wallpaper lives at
`configs/hypr/wallpapers/tahoe.png`; it was generated specifically for
HyprSequoia and does not reproduce or redistribute an Apple wallpaper. The
editable SVG fallback remains at `configs/hypr/wallpapers/default.svg`.

## Tahoe alpine lake

- Created: July 2026 with OpenAI's image-generation tool
- Prompt brief: an original 16:9 Sierra alpine lake at sunrise, with cool
  blue/cyan water, lavender and coral light, quiet sky for a translucent menu
  bar, and a darker foreground behind the Dock; no logos, UI, text, or marks
